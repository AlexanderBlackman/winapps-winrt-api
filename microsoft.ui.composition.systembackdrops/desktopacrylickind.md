## -description

Defines constants that specify what variant of Acrylic material is used.

## -enum-fields

### -field Default: 0

The system determines which Acrylic variant is used.

### -field Base: 1

The base Acrylic variant is used.

### -field Thin: 2

The thin Acrylic variant is used.
