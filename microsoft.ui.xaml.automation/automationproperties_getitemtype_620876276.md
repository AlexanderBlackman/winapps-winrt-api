# Microsoft.UI.Xaml.Automation.AutomationProperties.GetItemType

## -description
Gets the value of the [AutomationProperties.ItemType](automationproperties_itemtype.md) attached property for the specified [DependencyObject](../microsoft.ui.xaml/dependencyobject.md).

## -parameters
### -param element
The [DependencyObject](../microsoft.ui.xaml/dependencyobject.md) to check.

## -returns
The item type of the element.
