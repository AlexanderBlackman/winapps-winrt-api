# Microsoft.Windows.System.Power.PowerSupplyStatus

<!--

## -description

Defines values that represent the power supply status of the device.

## -enum-fields

### -field NotPresent: 0

Power supply is not present.

### -field Inadequate: 1

Power supply is not adequate.

### -field Adequate: 2

Power supply is adequate.
